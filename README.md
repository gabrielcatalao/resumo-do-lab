# Resumo do LAB.

Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO.

Durante a aula de laboratório foi solicido que criassemos uma conta no Microsoft Azure e que explorassemos a plataforma. Nos foi passado informações básicas, como trocar de idioma, trocar o tema e onde podiamos navegar pelas ferramentas disponibilidas pelo Azure.

## Sobre as aulas de introdução

Nos tópicos de introdução ao curso de Azure, foram levantados alguns temas como: 

### Modelo de Nuvens, podendo ser:
- Nuvens Publicas (Organização paga apenas o que consumir, o responsavel é o provedor)
- Nuvens Privadas (Responsabilidade da organização)
- Nuvens Hibridas (Cria uma cominicação entre as nuvens privadas e públicas)

*Também existe o modelo multicloud, mas não foi um tópico muito abordado pois não cai na prova da certificação*

### CapEx e OpEx
- Despesas de capital (CapEx)
  Gasto em infraestrutura fisica, valores tendem a diminuir com o tempo.
- Despesas Operacionais (OpEx)
  Gastos em produtos e serviços sob demanda, pagamento conforme uso. Cobrança imediata.
